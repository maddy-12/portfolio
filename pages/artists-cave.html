Projet Php avec un model MVC : Artists cave

Retour aux projets

Qu'est ce qu'Artists cave ?

Artists cave est un site qui repetorie les artistes illustrateurs en affichant leurs oeuvres ainsi que leurs coordonées. Il permet à l'utilisateur de s'inscrire, se connecter et créer des posts qui s'affichent dans la page d'acceuil du site.

Le projet a été versionné sur Github puis deployé sur l'hebergeur de sites gratuit Heroku.

Lien vers le repository Github

Technologies

Artists Cave est développé entièrement avec une programmation orientée objet (POO) avec PHP 7.4 et implémente le design pattern MVC (Model–view–controller).

Quant à la persistance des donnée, elle a été gérer grâce à un ORM (Object-relational mapping) s'appuyant sur des fichiers en format JSON.

Enfin, la partie front-end a été developpé en HTML 5, CSS 3 et Bootstrap 4.

Fonctionnalités

interface d'inscription

Inscription

Elle permet à l'utilisateur de s'inscrire dans le site avec un email, un username qui doit avoir au moins 4 caractères et un mot de passe possédant au moin 8 caractère et qui est par la suite haché et crypté grâce à la fonction MD5.

Connexion

Une fois inscrit, l'utilisateur va pouvoir se connecter avec ses identifiants qui seront vérifiés afin de s'assurer qu'ils correspondent aux données enregistrées dans la base de données.

interface de connexion

Page d'accueil

Inscription

La page d'accueil du site est visible à tous les utilisateurs, même ceux qui ne sont pas inscrits dans le site.

On y trouve les posts des artistes contenant leur les images de leurs oeuvres, le titre, la description, le type, ainsi que les prix de celles-ci. Enfin on y trouve un boutton qui permet de contacter l'artiste.

Création d'un nouveau post

Cette fonctionnalité est exclusivement réservée aux utilisateurs inscrits dans le site, c'est-à-dire les artistes.

La création d'un nouveau post se fait à l'aide d'un formulaire où l'utilisateurs doit renseigner l'url de l'image de son oeuvre, son titre, une brève déscription de cette oeuvre, son prix et enfin le type (ex: Aquarelle, peinture à l'huile...etc.).
