Qu'est ce qu'Open Museum ?

Open Museum est un site qui repertorie les musées de la région PACA. On y retrouve donc les coordonées de chaque musée provenant d'une base de donnée publique(Open data) mais qui ont été ensuite transférés sur une base de donnée no-sql.

Lien vers le repository Github

Technologies

Le projet à été développé avec la stack MEAN, c'est-à-dire:
  • MongoDB: une base de donnée non-relationnelle dans laquelle ont été transférés les données provenant d'une base de donnée publique.
  • Express.js: Un module du framework Node.js.
  • Angular 8 : Un framework front-end basé sur le language TypeScript.
  • Node.JS: Un framework back-end basé sur le language JavaScript.
Ainsi que Leaflet, une librairie permettant d'intégrer une carte géographique dans un site web.

Fonctionnalités

Page d'accueil

Sur cette page, l'utilisateur peut visualisé la liste des musées de la région PACA. Chaque carte contient le nom du musée ainsi que la ville où il se trouve.

Pour avoir plus de détails, l'utilisateur devra alors cliquer sur le lien "En savoir plus" situer en bas à gauche de la carte.

Détail d'un musée

Détail d'un musée

Lorsque l'utilisateur clique sur "En savoir plus" il est redirigé vers cette page ou il peut trouver les information du musée concerné en détail. C'est-à-dire, le nom du musée, son site web, son numéro de téléphone, ses horaires d'ouvertures ainsi que son adresse exacte accompagnée par l'emplacement de celle-ci sur la carte géographique.
